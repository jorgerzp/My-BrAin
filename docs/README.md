# Memoria del Proyecto Integrado — My_BrAIn

## Documento principal

- **`Memoria_MyBrAIn.docx`** — memoria en Word (capítulo 1 completo + índice plantilla + apartados 2–7 pendientes).

## Regenerar el Word

```bash
cd docs
npm run memoria
```

## En Microsoft Word

1. Abre `Memoria_MyBrAIn.docx`.
2. Completa la **portada** (nombre, centro, fecha).
3. **Referencias → Tabla de contenido → Tabla automática** para numerar páginas (usa estilos Título 1/2/3).
4. Añade **encabezado** (título del proyecto) y **pie** (“Página X de Y”) según normas del centro.
5. Inserta **capturas** en el apartado 1.9.
6. Ve ampliando los capítulos 2–7 en los apartados marcados como *[Pendiente…]*.
